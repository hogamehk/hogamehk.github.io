iPhone / iPad如何兌換美國 iTunes Gift Card（簡單教學）

  1. 使用美國 Apple ID 登入 App Store 或 iTunes Store。
  2. 點擊右上角頭像 → 選擇「兌換禮品卡或代碼」。
  3. 輸入收到的禮品卡序號（請勿加入多餘空格）。
  4. 確認無誤後提交，即可成功兌換至帳戶餘額。
  5. 兌換成功後，餘額會即時顯示在 Apple ID 帳戶中。

小提醒： 請確保 Apple ID 地區為美國，輸入序號時請勿加入多餘空格。

美國 iTunes Gift Card 常見問題（FAQ）

美國 iTunes Gift Card 可以在香港或台灣的 Apple ID 兌換嗎？

不可以。美國 iTunes Gift Card 僅限美國地區 Apple ID / US App Store 使用，香港、台灣、中國大陸或其他地區的 Apple ID 無法成功兌換。

Mac如何兌換美國 iTunes Gift Card？

開啟電腦上的 App Store App 或 Apple Music App → 點擊左下角的個人姓名或「登入」按鈕 → 點選右上角的 Redeem Gift Card（兌換禮品卡） →輸入 16 位禮品卡代碼，點擊 Redeem 完成充值。

美國 iTunes Gift Card 兌換後可以轉給別人使用嗎？

禮品卡序號本身可以轉讓，但兌換後的餘額會綁定該 Apple ID，無法轉移到其他帳戶。建議只給信任的親友使用，卡密售出後恕不退換。

購買後多久會收到禮品卡序號？

付款成功後 1~5 分鐘內，禮品卡序號會自動發送到您的 WhatsApp、WeChat 或 Line，即買即發。

美國 iTunes Gift Card 可以用來買什麼？

可在 US App Store 購買 App、遊戲內購、Apple Music 訂閱、iCloud 儲存空間、Apple TV+ 等服務。無法用於購買 Apple 硬件產品。

美國 iTunes Gift Card 有有效期嗎？

禮品卡序號本身通常有較長有效期，兌換後的 Apple ID 餘額一般無固定有效期，建議盡快使用。
