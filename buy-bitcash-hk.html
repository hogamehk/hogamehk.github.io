如何兌換/註冊 BitCash？（詳細步驟教學）

每張訂單我們免費附送詳細中文圖文教學，以下是最新正確流程：

1. 如何直接兌換/支付 (無需註冊)
如果你只是想在支持 BitCash 的網站（如日本遊戲、DMM、DLsite 等）消費，可以直接使用購買到的 16 位平假名 ID (ひらがなID)：

  • 選擇支付方式：在結帳頁面選擇 BitCash。
  • 輸入平假名 ID：在跳出的支付視窗中輸入你購買到的 16 位平假名。
  • 確認支付：系統會自動扣除對應點數。若餘額不足，可輸入多組 ID 進行合併支付。

2. 如何註冊 BitCash 會員 (BitCash Members)
註冊會員可以管理多組點數、享受積分回饋（びっ得コイン）及參與限定活動。

  • 第一步： 進入註冊頁面 → 前往 BitCash 官方註冊頁面並輸入電子信箱。
  • 第二步： 點擊驗證連結 → 在收到郵件後，點擊其中的 URL 進入正式註冊表單。
  • 第三步： 填寫會員資料
    • 暱稱 (Nickname)：15 字以內的平假名、片假名或英數。
    • 會員 ID & 密碼：設定登入用的 ID (6-25位) 與密碼 (7-25位，需包含英數)。
    • 安全性設定：設定「秘密問題與答案」，用於找回密碼。
  • 第四步： 完成二步驗證 (選填) → 系統會建議綁定電話號碼或副信箱以增強帳號安全性。

3. 如何將點數儲值進會員帳號 (My BitCash)
註冊成功後，你可以將實體卡或電子碼儲值到帳號中：

  • 登入 BitCash 會員網站。
  • 點擊「殘高チャージ」（儲值餘額）。
  • 輸入 16 位平假名 ID，點數即會轉換為 My BitCash 餘額，此後登入帳號即可一鍵支付。

小提醒：新手若卡在任何步驟，請即時 WhatsApp 我們，我們會一步一步即時指導（5–10 分鐘搞掂）。
